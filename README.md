# Not Another Boilerplate

A minimal React + TypeScript + Vite starter with React Query and Zustand. Uses Emotion for styling (no external CSS files).

Getting started (recommended with pnpm):

```bash
pnpm install
pnpm dev
```

Scripts:
- dev: start Vite dev server
- build: production build
- preview: preview production build
- type: run TypeScript type check

Notes:
- The example react-query hook fetches a todo from jsonplaceholder.typicode.com. Replace with your API.
- Styling is done with @emotion/react and @emotion/styled — no .css files.
